## 1. 启动程序

### 操作步骤

根据您的操作系统，双击对应的可执行文件启动程序：

| 平台 | 文件名称 |
| --- | --- |
| Windows | `加单分配系统_v3_Windows.exe` |
| macOS (ARM) | `加单分配系统_v3_Mac-ARM.app` |
| macOS (Intel) | `加单分配系统_v3_Mac-Intel.app` |
| Linux | `加单分配系统_v3_Linux` |

### 预期界面

程序启动后，您将看到以下主界面：

![主界面](screenshot_01_main.png)

**界面组成：**
- **标题区**：显示"加单商品分配系统"标题
- **分配逻辑说明**（可折叠）：深蓝色标题栏，点击可展开/折叠
- **文件选择区**：包含文件路径输入框和"浏览"按钮
- **按钮区**："执行加单分配"和"保存结果"按钮（初始状态为灰色不可点击）
- **状态区**：显示"等待文件选择..."
- **结果预览区**：显示分配结果表格

## 2. 选择Excel文件

### 操作步骤

1. 点击"浏览"按钮：

![点击浏览按钮](screenshot_02_browse.png)

2. 在弹出的文件选择对话框中，选择包含以下工作表的Excel文件：
   - `库存`：库存数据
   - `销售`：销售数据
   - `卖场等级`：卖场等级数据
   - `加单数量`：需要分配的商品数量

3. 选择文件后，文件路径会显示在输入框中：

![文件已选择](screenshot_03_file_selected.png)

**注意：**
- 支持的文件格式：`.xlsx`、`.xlsm`
- 文件必须包含所有四个工作表，否则会提示错误

### 预期结果

- "执行加单分配"按钮变为深绿色（可点击状态）
- 状态提示变为"文件已选择，点击执行开始分配"

---

## 3. 执行加单分配

### 操作步骤

1. 点击深绿色的"执行加单分配"按钮：

![点击执行按钮](screenshot_04_run.png)

2. 等待分配过程完成，期间状态会显示：
   - "正在读取Excel文件..."
   - "正在执行分配逻辑..."

### 预期结果

- 分配完成后弹出提示框："加单分配完成！"
- "保存结果"按钮变为深蓝色（可点击状态）
- 状态提示变为"分配完成！可以保存结果"
- 结果预览区显示分配结果的前20行

## 4. 查看分配结果

### 分配数量表格

结果预览区显示的第一部分是分配数量：

![分配结果预览](screenshot_05_result.png)

**表格说明：**
- **行**：各个卖场
- **列**：各个SKU
- **值**：分配给该卖场的商品数量

### 分配原因表格

系统会在Excel输出中生成第二个工作表"分配原因"：

| 分配原因 | 说明 |
| --- | --- |
| 断码修复(N) | 为确保卖场达到最低库存要求而分配，N为分配数量 |
| 销量匹配(N) | 根据30天销量匹配分配，N为分配数量 |
| 销尽率优先(N) | 按销尽率降序分配给B/C/D/OL级卖场，N为分配数量 |
| 剩余分配(N) | 将剩余商品分配给所有卖场，N为分配数量 |

**示例：**
```
卖场      SKU001
Store_A   断码修复(1),剩余分配(6)
Store_B   
Store_C   销尽率优先(13)
```

## 5. 保存分配结果

### 操作步骤

1. 点击深蓝色的"保存结果"按钮：

![点击保存按钮](screenshot_06_save.png)

2. 在弹出的保存对话框中，选择保存位置并输入文件名

### 预期结果

系统生成一个包含两个工作表的Excel文件：
- **分配数量**：包含每个卖场每个SKU的分配数量
- **分配原因**：包含每个分配的具体原因

## 6. 分配逻辑说明

### 展开/折叠操作

点击深蓝色标题栏可以展开或折叠分配逻辑说明：

![展开分配逻辑](screenshot_07_logic_expanded.png)

### 四个分配阶段

系统按以下优先级顺序执行分配：

| 优先级 | 阶段 | 说明 |
| --- | --- | --- |
| P1 | 断码修复 | 确保卖场达到最低库存要求 |
| P2 | 销量匹配 | 根据30天销量分配商品 |
| P3 | 销尽率优先 | 按销尽率降序分配（B/C/D/OL级） |
| P4 | 剩余分配 | 将剩余商品分配给所有卖场 |

## 7. 常见问题

### Q1: 程序启动后按钮都是灰色的？

**A1:** 这是正常现象。按钮需要在选择文件后才会变为可点击状态。请先点击"浏览"按钮选择Excel文件。

### Q2: 执行分配时提示错误？

**A2:** 请检查：
- Excel文件是否包含所有四个工作表
- 工作表是否包含必要的字段
- 数据格式是否正确（数值字段应为数字）

### Q3: 保存的Excel文件在哪里？

**A3:** 保存位置由您在保存对话框中选择。建议保存在容易找到的位置，如桌面。

### Q4: 程序运行很慢？

**A4:** 首次启动可能需要加载所有依赖项，稍慢是正常的。后续启动会更快。
