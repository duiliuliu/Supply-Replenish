## 其他平台打包方法

### Windows 平台打包

#### 环境要求
- Windows 10/11
- Python 3.8+
- 已安装 [pyinstaller](https://www.pyinstaller.org/)

#### 打包步骤

1. 将以下文件复制到Windows电脑：
   - `allocation_app.py`
   - `allocation_core.py`
   - `requirements.txt`
   - `pyinstaller_build.py`

2. 安装依赖
   ```bash
   cd /path/to/v3
   pip install -r requirements.txt
   pip install pyinstaller
   ```

3. 执行打包
   ```bash
   python pyinstaller_build.py
   ```

4. 获取打包结果
   - 可执行文件位于：`dist/加单分配系统_v3_Windows.exe`

## 注意事项

1. **文件大小**：打包后的文件约为 50-100MB（包含所有依赖）
2. **首次启动**：首次启动可能较慢，请耐心等待
3. **防火墙**：如果有防火墙提示，请允许网络访问（仅本地）
4. **跨平台兼容性**：Windows打包的程序只能在Windows运行，反之亦然
